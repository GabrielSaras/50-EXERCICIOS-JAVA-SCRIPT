## 🎯 Objetivo

Este projeto tem como objetivo praticar e consolidar os principais conceitos da linguagem JavaScript, incluindo:

- Operadores aritméticos
- Operadores lógicos e relacionais
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Manipulação de arrays e strings
- Arrow Functions
- Callbacks e Higher-Order Functions

---

## 🧠 Sobre os Exercícios

A lista é composta por 50 exercícios divididos em categorias:

### 🔹 Estruturas Condicionais
Exercícios envolvendo decisões como:
- Par ou ímpar
- Sistema de login
- Verificação de triângulo
- Cálculo de IMC

### 🔹 Estruturas de Repetição
- Laços `for`, `while` e `do...while`
- Tabuada
- Fibonacci
- Fatorial

### 🔹 Funções
- Funções com parâmetros e retorno
- Funções recursivas
- Funções com valores padrão

### 🔹 Arrays e Strings
- Métodos como `.map()`, `.filter()`, `.reduce()`
- Manipulação de strings
- Ordenação e busca

### 🔹 Conceitos Avançados
- Arrow Functions
- Callbacks
- Closures
- Currying
